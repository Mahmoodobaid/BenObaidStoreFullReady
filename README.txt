مشروع متجر بن عبيد - Android

هذا المشروع يحتوي على جميع الميزات التالية:

1. واجهات التطبيق:
   - SplashScreen: شاشة البداية بشعار المتجر.
   - ProductListScreen: قائمة المنتجات المخزنة.
   - AddProductScreen: نموذج لإضافة منتج جديد مع صور.
   - AboutScreen: معلومات التواصل وزر مزامنة البيانات.

2. إدارة قاعدة البيانات:
   - Room Database تستخدم لتخزين المنتجات محليًا.
   - دعم تخزين عدة صور لكل منتج (مسارات الصور فقط).

3. الميزات المضافة:
   - التقاط الصور من الكاميرا أو المعرض وربطها بالمنتج.
   - تحسين صورة المنتج باستخدام فلتر بسيط داخلي.
   - زر "مشاركة" لمشاركة بيانات المنتج عبر واتساب أو بلوتوث.
   - تسجيل الدخول بحساب Google باستخدام Google Sign-In.
   - مزامنة قاعدة البيانات مع Google Drive (رفع نسخة احتياطية).
   - استرجاع قاعدة البيانات من Google Drive (في حال إعادة التثبيت).
   - واجهة Bottom Navigation للتنقل بين الشاشات.

4. الأزرار في الواجهة:
   - "تحسين الصورة": تحسين صورة المنتج في AddProductScreen.
   - "مشاركة": مشاركة بيانات المنتج في ProductListScreen.
   - "مزامنة البيانات مع Google Drive": ترفع النسخة الحالية من قاعدة البيانات.
   - يمكن إضافة زر استرجاع عند الطلب (موجود في الكود).

5. تنبيهات مهمة:
   - الحد الأدنى للإصدار: Android 8.0 (minSdk = 26).
   - يجب تفعيل صلاحيات الإنترنت والكاميرا.
   - أول مرة يجب تسجيل الدخول بحساب Google قبل استخدام Google Drive.
   - يتم حفظ النسخة الاحتياطية في appDataFolder الخاص بالتطبيق على Google Drive.

6. خطوات إنشاء APK:
   - افتح Android Studio.
   - File > Sync Project with Gradle Files.
   - Build > Build APK(s).
   - سيظهر ملف APK في: app/build/outputs/apk/debug/app-debug.apk

تم إعداد هذا المشروع باستخدام Kotlin DSL + Jetpack Compose + Room + Google APIs.
